# 💫 About Me:

- 👨‍💻 I'm **Tran Hoang Hai**, also known as **BinkaDev**.
- 🎓 I'm a **Software Engineering** student at **Posts and Telecommunications Institute of Technology (PTIT)** **(2022 - 2027)**.
- ⚙️ My main direction is **Backend Engineering**.

- 🧠 I care about **clean architecture, readable code, database design, API consistency, real business logic, and strong project presentation for CV**.
